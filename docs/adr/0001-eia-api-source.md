# ADR 0001: 환경영향평가 공공 데이터 소스 — data.go.kr (공공데이터포털)

**상태**: Accepted
**작성일**: 2026-04-24
**결정자**: humpie_0413 (프로젝트 오너)

## 배경

`eia-workbench` 는 환경영향평가 실무 지원을 위해 환경현황 데이터(대기질·수질·소음 등), 평가 절차 메타데이터(협의 현황·결정내용 등), 유사 사례 검색용 공람 데이터가 필요하다. 후보 소스 검토:

- **EIASS** (`www.eiass.go.kr`) — 환경영향평가 정보지원시스템. 실제로는 포털/검색 UI 만 제공한다. 공식 OpenAPI 없음. 크롤링은 CLAUDE.md §2-4 (원문 재호스팅 금지) 에 저촉될 위험.
- **data.go.kr** (공공데이터포털) — 한국환경연구원 (KEI, 구 한국환경정책평가연구원) 이 환경영향평가 관련 OpenAPI 20종을 공식 공개 채널로 제공.

## 결정

환경영향평가 공공 데이터는 **data.go.kr 경유** 로 취득한다. EIASS 는 사용자에게 원문 페이지를 안내하는 **deep-link 용도로만** 참조한다 (재호스팅 아님).

## 근거

1. **공식 창구**: data.go.kr 는 정부 공식 OpenAPI 창구이며 KEI 가 직접 제공하므로 출처·최신성 보장.
2. **§2-4 준수**: EIASS 원문을 재호스팅/DB 저장하지 않고, 공공데이터포털 메타데이터 + EIASS deep-link 조합으로 운영.
3. **라이선스**: 대부분 KOGL 제1유형 (출처 표시 시 자유이용). MVP 에서는 메타데이터·링크 활용 중심이라 저작권 위험 낮음. 데이터셋별 개별 확인 필요.
4. **인증 단일화**: 공공데이터포털 일반 인증키 (`serviceKey` query param) 하나로 20종 전부 접근 가능. Cloudflare Secret 주입 용이.
5. **§2-2 준수**: 유료 LLM/상용 DB API 아님. 무료 공공 데이터.

## 인증

- **방식**: 공공데이터포털 일반 인증키 (HTTPS query parameter `serviceKey`)
- **환경변수**: `SERVICE_KEY` (Cloudflare Pages Secret 으로 주입)
- **발급 절차**: https://www.data.go.kr 회원가입 → 각 데이터셋 활용신청 → 마이페이지 → 개발계정
- **주입 명령**: `wrangler pages secret put SERVICE_KEY`
- **보안**: 키 값은 로그/에러 메시지/커밋에 절대 출력하지 않는다 (§6 `.env`/업로드 파일 커밋 금지와 동일).

## 라이선스

- 대부분 KOGL 제1유형 (출처 표시 시 자유이용, 상업적 이용 포함).
- 데이터셋별 상이하므로 `packages/eia-data/src/endpoints/<dataset>.ts` 작성 시 활용신청 페이지의 라이선스를 개별 확인한다.
- 사용자에게 표시할 때는 "출처: 공공데이터포털 (data.go.kr), 제공: 한국환경연구원" 표기를 유지한다.

## 제약

- **일 호출 한도**: 개발계정 기준 데이터셋별 상이 (일반적으로 1,000건/일). 운영계정 전환 시 심사 필요. 클라이언트 측 캐시·레이트리밋 필수.
- **응답 포맷**: XML/JSON 혼재. 엔드포인트별 파싱 로직 분기 필요.
- **공통 응답 envelope**: `response.header.resultCode` (정상 = `'00'`), `response.header.resultMsg`, `response.body.items.item[]`.
- **가용성**: 정부 시스템 특성상 점검/장애 가능. 재시도 + 타임아웃 + graceful degradation 필요.

## 결과

- 신규 패키지 `packages/eia-data/` 로 data.go.kr 클라이언트 스캐폴딩 (본 PR).
- 실제 엔드포인트 구현은 다음 feature (유사사례 검색) 에서 필요 데이터셋부터 채움.
- EIASS 는 `packages/eia-data/src/deep-link.ts` 를 통해 원문 페이지 URL 생성 용도로만 사용.
- CLAUDE.md §3 기술 스택에 "환경영향평가 공공 데이터: data.go.kr (KEI 제공). SERVICE_KEY (Cloudflare Secret)" 한 줄 추가.

## 대안 (기각)

- **대안 A — EIASS 크롤링**: §2-4 재호스팅 원칙 위반 위험. 공식 API 아니므로 호환성 보장 없음. **기각**.
- **대안 B — 유료 상용 API**: §2-2 유료 API 금지. **기각**.
- **대안 C — 자체 데이터셋 구축**: MVP 범위 밖. 데이터 수집·갱신 비용 부담. **기각**.
